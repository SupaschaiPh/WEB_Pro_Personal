Bootstrap ชุดเครื่องมือ CSS ช่วยให้นักพัฒนาสร้างเว็บไซต์และแอปพลิเคชันเว็บได้อย่างยอดเยี่ยม

ที่มา : https://getbootstrap.com/ ที่มา : https://en.wikipedia.org/wiki/Bootstrap_(front-end_framework)

Bootstrap เป็นเฟรมเวิร์ค CSS โอเพ่นซอร์ส ฟรี ที่มุ่งเน้นการพัฒนาเว็บไซต์แบบ responsive และ mobile-first ประกอบด้วยเทมเพลตการออกแบบบนพื้นฐาน HTML, CSS และ JavaScript สำหรับองค์ประกอบส่วนต่อประสานผู้ใช้ต่างๆ เช่น ตัวอักษร ฟอร์ม ปุ่ม เมนู นำทาง

ความนิยมของ Bootstrap

  • ณ เดือนพฤษภาคม 2023 Bootstrap ติดอันดับโปรเจกต์ที่มียอดดาวมากที่สุดเป็นอันดับ 17 บน GitHub (อันดับ 4 สำหรับไลบรารี) โดยมียอดดาวเกิน 164,000 ดาว
  • ตามข้อมูลของ W3Techs เว็บไซต์ 19.2% ทั่วโลกใช้ Bootstrap

Bootstrap คืออะไร ?

Bootstrap เป็นไลบรารีที่ใช้กับ HTML, CSS, และ JavaScript โดยมีการใส่ใจในการทำให้การพัฒนาหน้าเว็บที่เต็มไปด้วยข้อมูลเป็นเรื่องง่ายขึ้น (ต่างจากการพัฒนาแอปพลิเคชันเว็บ) จุดประสงค์หลักของการเพิ่ม Bootstrap เข้าสู่โปรเจคเว็บคือเพื่อนำสี ขนาด แบบอักษร และเค้าโครงที่ Bootstrap ได้กำหนดไว้มาปรับใช้ในโปรเจคนั้น ๆ ดังนั้น, ปัจจัยหลักคือว่านักพัฒนาที่รับผิดชอบโปรเจคพบว่าเลือกนี้เป็นที่ชอบของตน

เมื่อถูกเพิ่มเข้าไปในโปรเจค, Bootstrap จะให้คำนิยามสไตล์พื้นฐานสำหรับทุกๆ องค์ประกอบ HTML ผลลัพธ์คือมีลักษณะเดียวกันสำหรับข้อความ, ตาราง, และองค์ประกอบแบบฟอร์มทั่วไปที่ปรากฎในเว็บบราวเซอร์ทั้งหมด นอกจากนี้, นักพัฒนาสามารถใช้คลาส CSS ที่ถูกกำหนดไว้ใน Bootstrap เพื่อปรับแต่งลักษณะของเนื้อหาได้อีก ตัวอย่างเช่น, Bootstrap ได้เตรียมการให้ใช้งานกับตารางที่มีสีสว่างและสีเข้ม, หัวเรื่องหน้า, การเน้นคำพูดที่สำคัญมากขึ้น, และข้อความที่มีการเน้นพิเศษ

Bootstrap ยังมี JavaScript components หลายตัวที่ไม่ต้องการไลบรารีอื่น ๆ เช่น jQuery ซึ่งเสริมสร้างองค์ประกอบส่วนหน้าผู้ใช้เพิ่มเติม เช่น กล่องโต้ตอบ, คำแนะนำเคล็ดลับ, แถบความคืบหน้า, เมนูลงจากด้านบน, และการสลับสไลด์ ทุก component ของ Bootstrap ประกอบด้วยโครงสร้าง HTML, การประกาศ CSS, และบางกรณีทำตามด้วยโค้ด JavaScript ที่เกี่ยวข้อง นอกจากนี้, มีการขยายฟังก์ชันของบางองค์ประกอบส่วนหน้าผู้ใช้ที่มีอยู่แล้ว, เช่น ฟังก์ชันออโต้คอมพลีตสำหรับช่องป้อนข้อมูลเป็นต้น

องค์ประกอบที่โดดเด่นที่สุดของ Bootstrap คือ layout components ซึ่งมีผลต่อหน้าเว็บทั้งหมด องค์ประกอบหลักที่เกี่ยวข้องกับการจัดวางคือ "Container" โดยที่ทุกองค์ประกอบอื่น ๆ ในหน้าเว็บจะถูกวางไว้ใน Container นักพัฒนาสามารถเลือกระหว่าง fixed-width container และ fluid-width container ในขณะที่ fixed-width container จะมีขนาดคงที่และใช้หนึ่งในขอบเขตความกว้างที่กำหนดไว้ล่วงหน้าห้าขนาดต่าง ๆ ขึ้นอยู่กับขนาดของหน้าจอที่แสดงหน้าเว็บ, fluid-width container จะเติมความกว้างของหน้าเว็บเสมอ ๆ

  • น้อยกว่า 576 pixels
  • 576–768 pixels
  • 768–992 pixels
  • 992–1200 pixels
  • มากกว่า 1200 pixels

เมื่อมี Container อยู่แล้ว, องค์ประกอบการจัดวาง Bootstrap อื่น ๆ ใช้ CSS Flexbox layout โดยการกำหนดแถว (rows) และคอลัมน์ (columns) ได้

เวอร์ชันที่ถูกคอมไพล์ไว้ของ Bootstrap มีให้ในรูปแบบของไฟล์ CSS 1 ไฟล์ และ JavaScript 3 ไฟล์ ซึ่งสามารถเพิ่มเข้าไปในโปรเจคได้ทันที. แต่รูปแบบที่ยังไม่ได้ถูกคอมไพล์ (raw form) ของ Bootstrap ช่วยให้นักพัฒนาสามารถทำการปรับแต่งเพิ่มเติมและปรับปรุงขนาดได้. รูปแบบนี้เป็นแบบโมดูล, นั่นคือนักพัฒนาสามารถลบออกส่วนประกอบที่ไม่จำเป็น, ใช้ธีม, และปรับแก้ไขไฟล์ Sass ที่ยังไม่ได้คอมไพล์ได้ตามต้องการ.

ประวัติและพัฒนาการของ Bootstrap

ต้นกำเนิดเริ่มต้น

Bootstrap ซึ่งตั้งชื่อเดิมว่า Twitter Blueprint, ถูกพัฒนาโดย Mark Otto และ Jacob Thornton ที่ Twitter เป็นเฟรมเวิร์กเพื่อส่งเสริมความสม่ำเสมอในการใช้งานที่เกี่ยวข้องกับเครื่องมือภายใน. ก่อนที่จะมี Bootstrap, มีการใช้หลายไลบรารีสำหรับการพัฒนาอินเตอร์เฟซที่ทำให้เกิดความไม่สมดุลและภาระการบำรุงรักษาที่มีน้ำหนักมาก. ตามคำบรรยายของ Otto: "กลุ่มนักพัฒนาขนาดเล็กและฉันได้รวมตัวกันเพื่อออกแบบและสร้างเครื่องมือภายในใหม่และเห็นโอกาสที่จะทำบางสิ่งมากขึ้น. ผ่านกระบวนการนั้น, เราเห็นตัวเองกำลังสร้างบางสิ่งที่มีน้ำหนักมากกว่าเพียงเครื่องมือภายในอีก. หลายเดือนต่อมา, เราได้มีรุ่นแรกของ Bootstrap เป็นวิธีที่จะเอกสารและแบ่งปันแพทเทิร์นการออกแบบทั่วไปและทรัพยากรภายในบริษัท." หลังจากพัฒนาไปอีกไม่กี่เดือนโดยกลุ่มนักพัฒนาขนาดเล็ก, นักพัฒนามากมายที่ Twitter เริ่มมีส่วนร่วมในโครงการเป็นส่วนหนึ่งของ Hack Week, สัปดาห์แบบ hackathon สำหรับทีมพัฒนา Twitter. โครงการถูกเปลี่ยนชื่อจาก Twitter Blueprint เป็น Bootstrap และปล่อยเป็นโปรเจค open-source เมื่อวันที่ 19 สิงหาคม 2011. มันได้รับการบำรุงรักษาต่อมาโดย Otto, Thornton, กลุ่มนักพัฒนาหลักขนาดเล็ก, และชุมชนมากมายของผู้ร่วมพัฒนา.

Bootstrap 2

ในวันที่ 31 มกราคม 2012, Bootstrap 2 ได้ถูกปล่อยออกมา ซึ่งเพิ่มการสนับสนุนในตัวอักษรไอคอน (Glyphicons), องค์ประกอบใหม่หลายประการ, และการเปลี่ยนแปลงที่บางส่วนขององค์ประกอบที่มีอยู่. รุ่นนี้รองรับ responsive web design, ซึ่งหมายถึงเลย์เอาต์ของหน้าเว็บจะปรับตามโดยอัตโนมัติโดยพิจารณาถึงลักษณะของอุปกรณ์ที่ใช้ (ไม่ว่าจะเป็นเดสก์ท็อป, แท็บเล็ต, หรือโทรศัพท์มือถือ).

Bootstrap 3

ในวันที่ 19 สิงหาคม 2013, Bootstrap 3 ถูกปล่อย. มีการออกแบบใหม่ให้กับองค์ประกอบโดยใช้ flat design และเข้าใช้งานแบบ mobile first approach. Bootstrap 3 มีระบบปลั๊กอินใหม่ที่มีเหตุผลด้วยเหตุผลเนมสเปซ. Bootstrap 3 ลดการสนับสนุน Internet Explorer 7 และ Firefox 3.6 แต่มี polyfill ที่เลือกใช้สำหรับเบราว์เซอร์เหล่านี้.

Bootstrap 4

Mark Otto ประกาศ Bootstrap 4 เมื่อวันที่ 29 ตุลาคม 2014. เวอร์ชัน alpha แรกของ Bootstrap 4 ได้ถูกปล่อยให้ใช้งานเมื่อวันที่ 19 สิงหาคม 2015. เวอร์ชัน beta แรกได้ถูกปล่อยให้ใช้งานเมื่อวันที่ 10 สิงหาคม 2017. Otto ได้ระงับการทำงานกับ Bootstrap 3 เมื่อวันที่ 6 กันยายน 2016 เพื่อให้มีเวลาเพียงพอในการทำงานกับ Bootstrap 4. Bootstrap 4 ได้ถูกสมบูรณ์เมื่อวันที่ 18 มกราคม 2018.

การเปลี่ยนแปลงที่สำคัญรวมถึง:

  • การเขียนโค้ดใหม่ทั้งหมด
  • การแทนที่ Less ด้วย Sass
  • เพิ่ม Reboot, คือชุดของการเปลี่ยนแปลง CSS ที่เฉพาะตัวสำหรับองค์ประกอบในไฟล์เดียว, โดยใช้ Normalize เป็นพื้นฐาน
  • ยกเลิกการสนับสนุน IE8, IE9, และ iOS 6
  • การสนับสนุน CSS Flexible Box
  • เพิ่มตัวเลือกปรับแต่งการนำทาง
  • เพิ่มการให้บริการช่องว่างและขนาดที่ปรับได้ตามขนาดหน้าจอ
  • การเปลี่ยนจากหน่วย pixels ใน CSS เป็น root ems
  • เพิ่มขนาดฟอนต์โลกจาก 14px เป็น 16px เพื่อความอ่านง่ายขึ้น
  • การลดความจำเป็นของ panel, thumbnail, pager, และ well components
  • การลดออกไปจาก icon font ของ Glyphicons
  • จำนวน[ปริมาณ]มหาศาลของ utility classes
  • การปรับปรุงการทำสวยงามฟอร์ม, ปุ่ม, ตัวเลือกเมนูแบบ drop-down, อ็อบเจกต์สื่อ, และคลาสรูปภาพ

Bootstrap 4 รองรับเวอร์ชันล่าสุดของ Google Chrome, Firefox, Internet Explorer, Opera, และ Safari (ยกเว้นใน Windows). นอกจากนี้, มันยังสนับสนุนถอยหลังไปที่ IE10 และ Firefox Extended Support Release (ESR) ล่าสุด.

Bootstrap 5

Bootstrap 5 ได้ถูกเปิดตัวอย่างทางการเมื่อวันที่ 5 พฤษภาคม 2021.

การเปลี่ยนแปลงที่สำคัญรวมถึง:

  • คอมโพเนนท์เมนู offcanvas ใหม่
  • การลดความขึ้นอยู่กับ jQuery เพื่อใช้ vanilla JavaScript
  • การเขียนใหม่กริดเพื่อรองรับการรีสพอนซีวการเรียงและคอลัมน์ที่อยู่นอกตาราง
  • การย้ายเอกสารจาก Jekyll ไปที่ Hugo
  • ยกเลิกการสนับสนุน Internet Explorer
  • การย้ายโครงสร้างการทดสอบจาก QUnit ไปที่ Jasmine
  • เพิ่มชุดไอคอน SVG ที่กำหนดเอง
  • เพิ่มคุณสมบัติ CSS custom properties
  • API ที่ปรับปรุง
  • ระบบกริดที่ปรับปรุง
  • เอกสารการปรับแต่งที่ดีขึ้น
  • การอัปเดตแบบฟอร์ม
  • การสนับสนุน RTL
  • การสนับสนุน Darkmode ที่ซ่อนอยู่
