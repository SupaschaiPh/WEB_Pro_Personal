Internet Explorer บทเรียนสำคัญสำหรับเว็บเบราว์เซอร์ยุคใหม่

ที่มา : https://www.techtarget.com/searchenterprisedesktop/definition/Internet-Explorer

Internet Explorer (IE) เคยเป็นเว็บเบราว์เซอร์ยอดนิยมที่มาพร้อมกับระบบปฏิบัติการ Microsoft Windows แต่ปัจจุบันยกเลิกการสนับสนุนแล้วใน Windows 10 แล้ว แทนที่ด้วยเบราว์เซอร์ใหม่ของ Microsoft ชื่อ Edge Browser แม้ว่า IE จะยังคงใช้งานได้ในบางเวอร์ชันของ Windows เช่น Windows 8.1 และ Microsoft Edge ก็มีฟีเจอร์ที่เรียกว่าโหมด IE

ข้อมูลปี 2023 ระบุว่า Internet Explorer เป็นเว็บเบราว์เซอร์ที่ได้รับความนิยมน้อยที่สุดบนเดสก์ท็อป มีแชร์การตลาดเพียง 0.45% เมื่อเทียบกับ Google Chrome ที่มี 66.02% ตามข้อมูลของ StatCounter IE ครองตำแหน่งเบราว์เซอร์ยอดนิยมตั้งแต่ปี 1999 เมื่อแซงหน้า Netscape Navigator มาจนถึงปี 2012 ที่ Chrome ขึ้นนำ คู่แข่งอื่นๆ ได้แก่ Mozilla Firefox ซึ่งเป็นเบราว์เซอร์โอเพนซอร์สที่พัฒนาโดยใช้โค้ดจาก Netscape Navigator และ Safari ของ Apple iOS

อดีตอันรุ่งโรจน์และจุดจบของ Internet Explorer

Internet Explorer เคยเป็นเว็บเบราว์เซอร์อันดับหนึ่งของโลก เบื้องต้น Microsoft พัฒนา IE โดยอาศัยเทคโนโลยีจาก Spyglass เจ้าของเบราว์เซอร์ Mosaic บุกเบิกยุคแรก เวอร์ชันแรกเปิดตัวในปี 1995 สำหรับ Windows 95 ตามมาด้วยเวอร์ชัน 2 รองรับระบบความปลอดภัยและคุกกี้ เวอร์ชัน 3 ในปี 1996 เพิ่มฟีเจอร์ Java และ Cascading Style Sheets

ปี 1998 รัฐบาลสหรัฐฯ ฟ้องร้อง Microsoft ฐานผูกขาดตลาด โดยบังคับรวม IE กับ Windows จนปี 2001 ทั้งสองฝ่ายตกลงกันโดยไม่ต้องแยก IE ออก

ตลอดอายุขัย IE มีอัปเดต 11 เวอร์ชันและแพทช์นับไม่ถ้วนเพื่ออุดช่องโหว่ IE 11 เวอร์ชันสุดท้ายเปิดตัวในปี 2013 ถูกแทนที่ด้วย Microsoft Edge ใน Windows 10 ปี 2015

ปี 2020 Microsoft ประกาศยุติการสนับสนุนแอปพลิเคชัน Office สำหรับ IE ปี 2022 ยุติการสนับสนุน IE 11 อย่างเป็นทางการ กุมภาพันธ์ 2023 อัปเดต Microsoft Edge ปิดใช้งาน IE ถาวรใน Windows 10 ส่วนใหญ่ ผู้ใช้จะถูกเปลี่ยนไปใช้ Edge

ปัจจุบัน หากยังต้องการใช้ IE สำหรับเว็บเก่า สามารถเปิดใช้งานโหมด IE ใน Microsoft Edge เพื่อใช้งานเว็บยุคใหม่และเก่าภายในเบราว์เซอร์เดียวกัน นอกจากนี้ Microsoft ยังยุติการสนับสนุน IE ในระบบอื่นๆ อย่าง Unix และ Mac OS X อีกด้วย

ฟีเจอร์และการตั้งค่าความปลอดภัยของ IE

แม้ Microsoft จะยุติการสนับสนุน Internet Explorer แล้ว แต่การรู้จักฟีเจอร์และการตั้งค่าความปลอดภัยของมันก็ยังมีประโยชน์อยู่บ้าง นี่คือสรุปคร่าวๆ

ส่วนติดต่อผู้ใช้:

  • แถบที่อยู่ แสดงเว็บไซต์ที่เข้าชม
  • แถบเมนู หลากหลายฟังก์ชัน เช่น บันทึก เปลี่ยนขนาดตัวอักษร
  • แถบรายการโปรดักต์ เว็บไซต์ที่ใช้งานบ่อย
  • แถบสถานะ แสดงสถานะการโหลดหน้าเว็บ

เมนูหลัก:

  • File:บันทึก เปิดหน้าต่าง/แท็บใหม่ แก้ไขโค้ดหน้าเว็บ (บางเวอร์ชัน)
  • Edit:ตัด คัดลอก วาง ค้นหาในหน้าเว็บ
  • Tools:ฟีเจอร์สำคัญด้านความปลอดภัย เช่น บล็อกป๊อปอัพ โหมดไม่ระบุตัวตน การติดตามตำแหน่ง ActiveX filter

การตั้งค่าความปลอดภัย:

  • Internet Options:ปรับแต่งหน้าแรก ลบประวัติ เปลี่ยนธีม ตั้งระดับความปลอดภัย เปิด/ปิดโหมดป้องกัน
  • Security:เลือกระดับความปลอดภัย เปิด/ปิดโหมดป้องกัน
  • Privacy:บล็อกป๊อปอัพ โหมดไม่ระบุตัวตน ป้องกันเว็บไซต์ติดตามตำแหน่ง
  • Content:ใบรับรอง การกรอกข้อมูลอัตโนมัติ ฟีด ข่าวสาร
  • Connection:ตั้งค่าการเชื่อมต่ออินเทอร์เน็ต (ไม่ค่อยใช้ในปัจจุบัน)
  • Programs:จัดการ Add-on การเชื่อมโยงไฟล์ แอปพลิเคชันที่เกี่ยวข้อง
  • Advanced:ปรับแต่งเบราว์เซอร์อย่างละเอียด
